== Description ==

Stop Self Pingbacks and Trackbacks to own posts and pages. You can avoid unwanted selfpings and moderation alerts for your own posts or pages.

Visit Stop Self Ping <a href="http://www.joelsays.com/plugins/stop-self-ping/">official website</a>


Features of <a href="http://www.joelsays.com/plugins/stop-self-ping/" target="_blank">Stop Self Ping</a> are,

 * Automatically disables self pingbacks and trackbacks to own posts and pages.
 * Disables annoying trackback comment moderation message.
 * 24x7 <a href="http://www.joelsays.com/contact-me" target="_blank">Quick Support</a>
 * More optimized website

== Installation ==

1. Upload `Stop Self Ping` plug-in zip file to the `/wp-content/plugins/` directory
2. Alternately you can add plugin from wp dashboard itself.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That is all you have to do. Remaining will be done plugin.

Note : You can disable comments on pages or posts using default disqus options.
